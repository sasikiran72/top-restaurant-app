# Top Restaurants of the Week – Accessible Web App

## Overview
A responsive and accessible web application that showcases top restaurants with rankings, reviews, and accessibility features. Users can sign up, log in, and submit reviews.

Developed for **COMP 5970/6970 Assistive and Accessible Computing**.

---

## Links
- Live App: https://sasikiran72.github.io/top-restaurant-app/
- GitHub Repo: https://github.com/sasikiran72/top-restaurant-app

---

## Technologies
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- LocalStorage  

---

## Key Features
- Restaurant leaderboard with rankings and accessibility scores  
- Detailed restaurant pages with reviews  
- User sign-up and login system  
- Review submission (logged-in users only)  
- Profile page displaying user data and reviews  
- Accessibility features (high contrast, font size control, semantic HTML)  
- Fully responsive design

## Run Locally
1. Download or clone the repo  
2. Open in VS Code  
3. Run `index.html` using Live Server
